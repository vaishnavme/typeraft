## [Typeraft](https://typeraft.vercel.app/)

Typeraft is a modern WYSIWYG editor powered by Tiptap, offering a seamless and customizable writing experience.

![Typeraft screenshot](https://raw.githubusercontent.com/vaishnavme/typeraft/refs/heads/main/public/typeraft-og.png)

This project showcases a wide range of Tiptap extensions, including but not limited to:

- Bold, Italic, Underline, Strike-through
- Headings
- Lists (Bullet and Numbered)
- Code Block with syntax highlight and Copy button
- Blockquote
- Link
- Emoji
